# Python Mini Projects

Hi! I'm Umme Habeeba, a 1st year CSE (AI & ML) 
student. This repository contains some small 
Python projects I built while learning Python 
during my first year of college.



## What's inside

**Student Management System**
A simple program to add, view, search and 
delete student records. I used lists to 
store the data.

Quiz Application**
A small quiz game with 5 Python related 
questions. It shows your score at the end 
and tells you if you passed or not.

Expense Tracker**
A program to track your daily expenses. 
You can add expenses, view them by category 
and see your total spending.

Login System**
A basic register and login system. 
Usernames and passwords are saved in 
a text file.

How to run

Just install Python 3 and run whichever 
file you want:

python filename.py

No external libraries needed — just 
pure Python!
